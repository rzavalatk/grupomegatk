USER GUIDE

Having a Glamorous e-Commerce theme is one of the foremost necessities of being an e-commerce success. With Kinetik, you have got this covered.

Install Theme

Once you have bought the theme from the Odoo Apps store, you will notice that you have got two folders. The first one is ‘ks_theme_base’ and the second one is ‘ks_theme_kinetik’.

Step 1 : Place both of the folders with your other addons.

Step 2 : And search for the Kinetik theme.

Step 3 : Install the ‘ks_theme_kinetik’ and after it ‘ks_theme_base’ will be automatically installed.

Mega Menu

Menus play an important role in smooth navigation for the customer. As everything is available in the menu, the products, the brands, the categories, the menu are the roadmap of any e-Commerce website. Kinetik provides the option so that the customer can configure their menu and Mega Menu.

To enable your Mega Menu, you need to click on the menu Website Configuration Menus

NOTE: Not everyone can see the setting of the Mega Menu, it is mandatory to turn on the DEVELOPER MODE.

When a user clicks on the menu, they can create a new menu by filling in the option as these display as the Menu.

To make the menu as the Mega menu, check the checkbox 'Is Mega Menu’.

Mega Menu comes under the Parent Menu so choose the Parent under which the Particular Mega Menu has to be displayed.

Once the checkbox is clicked, a subform appears at the bottom providing three main options to customize the Mega menu.

The First Menu is the Child Menu. This Menu is to customize the Child menu, which means the Parent Menu has sub-menu which is known as Child menu. Update Screenshots

The second menu named Mega Menu Setup allows choosing what to be displayed in Mega menu
i.e., Products, Categories or Brands.

Users can select which types of menus they are looking for like products, brands, categories.

The third tab for customization is named Mega Menu Advance. Using the options in this menu, you can customize the Background Image of your Mega Menu and even set its Font Color.

The fourth option is called Add Slider/Image in Mega Menu.

This will help you to add Images or Sliders to the Menus.

It provides the option to add the Image with the Product Title, Link, Description, etc.

> Add a Slider

It provides you with an option to set a slider for either product, category or brand as your Mega Menu option.

This is how your Mega Menu looks when you are enabled to showcase images as well as sliders for products along with a normal Mega Menu Category display of products

Landing Page

For Using the ‘Landing Page’, the user needs go to the Kinetik Repository by following the path ’theme-kinetik/ks_theme_kinetik/static/landing_modules’ .

In the landing_modules folder, the user will find all the landing page modules. After that, you can just copy the path ’/theme-kinetik/ks_theme_kinetik/static/landing_modules’ and paste it to the ’odoo-server.conf’ file, for the module that you needs to see.

For example : ‘addons_path = /theme-kinetik/ks_theme_kinetik/static/landing_modules,‘.

Now the user need to Go to Website Configuration Settings

When a user clicks on the ’Settings’, you will be able to select the ‘Landing pages’ that you needs to use, as displayed in the form.

Previous and Next Product

When a User click on the ’Previous’ button on product detail page then it will redirect to the previous product detail page and If he click on ’Next’ button then it will redirect to the next product detail page.

Trending Products

Trending Products is a feature in which the User will view the list of the products that have been viewed by the user and those products will display in the Trending Products.

Contact Us Form

For Using the ‘Contact Us Form’, the user needs go to the Kinetik Repository by following the path ’theme-kinetik/ks_theme_kinetik/static/Other_modules’ .

In the Other_modules folder, the user will find the ks_contact_form. After that, you can just copy the path ’/theme-kinetik/ks_theme_kinetik/static/Other_modules’ and paste it to the ’odoo-server.conf’ file.

For example : ‘addons_path = /theme-kinetik/ks_theme_kinetik/static/Other_modules,‘.

Now the user need to Go to Apps Search for ks_contact_form Install

If the module is not found then the user needs to click on the Update Apps List after activating the developer mode.

Now when the user clicks on Contact Us in the website header, you will be able to see the contact us form.
