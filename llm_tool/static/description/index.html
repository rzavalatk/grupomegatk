About This Module

Boost your Odoo database automation with AI-powered assistants using ChatGPT, Grok, Anthropic, and DeepSeek. Streamline workflows, optimize data management, and enhance productivity with AI tools seamlessly integrated into your Odoo instance.

This module provides a robust framework for integrating Large Language Models (LLMs) with Odoo, enabling intelligent interactions through configurable tools. Ideal for AI-driven ERP management, it empowers businesses with an Odoo AI assistant to automate workflows and enhance business automation.

Key Features

  • Function Calling: Enable AI models to call specific functions based on user requests
  • Odoo AI Automation: Define and manage LLM tools with custom implementations
  • AI Assistants for Odoo: Dynamic schema generation from Pydantic models
  • Chat AI Integration: Flexible override options for tool descriptions and schemas
  • AI-Powered ERP: Integration with Odoo mail threads for intelligent chat interactions
  • Extensible Design: Add new tool implementations for your business needs

Installation

  1. Download the module from the Odoo Store or GitHub.
  2. Install dependencies: base, mail, llm.
  3. Upload and install the module in your Odoo instance.
