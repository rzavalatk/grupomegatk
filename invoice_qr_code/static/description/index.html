QR Code on Invoice

This Module will allow to Generate and Dsiplay QR Code for Invoice.

By using this module you can Generate QR Code for Invoice. Thae QR code will visible on Invoice form. You can also print the QR code on Invoice as well.

Features

  • This module is user friendly.
  • This module reduces human effort.
  • You can generate QR Code for Invoice.
  • You can Display the QR Code on Invoice records
  • Print the QR Code on the PDF reports as well
  • QR code will display the Information of Seller name, Vat Registration Number, Total with VAT, VAT total.
